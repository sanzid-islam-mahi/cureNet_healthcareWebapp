# CureNet Screenshot Placeholders

This folder contains placeholder screenshot assets for the README.

Purpose:
- keep the README structure ready for visual documentation
- give you stable file names now
- let you replace them later with real screenshots without rewriting the README

Current placeholders:
- `landing-page-placeholder.svg`
- `patient-dashboard-placeholder.svg`
- `doctor-appointments-placeholder.svg`
- `receptionist-queue-placeholder.svg`
- `admin-clinics-placeholder.svg`
- `medical-history-placeholder.svg`
- `notification-center-placeholder.svg`
- `medical-imaging-placeholder.svg`

Recommended replacement workflow:
1. capture a real screenshot
2. crop and optimize it
3. save it using the same file name
4. keep dimensions reasonably consistent

That way the README keeps working with no markdown changes.
